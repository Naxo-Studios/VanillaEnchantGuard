# 🔑 Permissions

Each Admin Panel has its own permission node:

```yaml
veg.admin - /veg admin
```

Perfect for:

- Unlimited Max Level Enchants
- Items Illegal in Shulkers/Enderchest/Inventory

---

# 🎨 MiniMessage Color Support

Use modern RGB hex colors directly inside yours Messages/Menu.

### Example

```yaml
title: "<gradient:#00ff99:#00ffff>VanillaEnchantGuard</gradient>"
```

---

# ⚙️ Installation

1. Download the plugin `.jar`
2. Move it to your server's `plugins/` folder
3. Restart the server
4. Configure your VanillaEnchantGuard
5. Enjoy ✨

---

# 📁 Configuration

Plugin folder:

```yaml
/plugins/VanillaEnchantGuard/
```

Included files:

```yaml
config.yml
menu.yml
```
